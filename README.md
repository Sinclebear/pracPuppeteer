# pracPuppeteer
puppeteer 사용 연습

## 실행

```
node puppet.js
```

### 제약사항

- `Airbnb Plus` 호스트의 경우 스크래핑 불가, `슈퍼호스트가 아닌` 경우에도 정상 동작하지 않음.
- 따라서, 현재 버전의 경우 유저가 수동으로 위 조건에 일치하는 URL 목록들을 부어줘야 함.
